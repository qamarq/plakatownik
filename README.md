# Posterownik

Projekt na laboratorium z języków skryptowych, rok akademicki 2025/2026.

Autorzy: Kamil Marczak, Konrad Guzek


Frontend Dev:

```bash
cd frontend
pnpm install
pnpm dev
```

Backend Dev
```bash
cd backend
uvicorn app.main:app --reload
```
